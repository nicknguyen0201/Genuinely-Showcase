# Website (Frontend)

This folder contains the frontend code for the Genuinely web application.

## Tech Stack
- **React** + **TypeScript**
- **Vite** for development and bundling
- **CSS** for styling

## Overview
The frontend handles all user-facing pages, including:
- Authentication and onboarding
- Personality & vibes survey
- Dashboard and match views
- Terms of use

It communicates with the backend via Supabase for authentication, data access, and row-level security, and integrates with email workflows for user notifications.
